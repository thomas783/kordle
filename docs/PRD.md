한글 **자모 단위 Wordle** 웹 게임. 카카오톡 미니게임 "단어 맞히기"가 하루 한 판 제한이라, 친구들과 무제한으로 즐기는 버전을 만든다.

- **대상**: 본인 + 가까운 친구 (소규모, 비공개)
- **플랫폼**: 모바일 우선, PC 호환 (반응형)

## 2. 게임 규칙 (핵심)

- **5자모 단어**를 맞힌다. "5자리"는 음절이 아니라 **자모로 분해했을 때 5칸**.
  - 예: `꽉` → `ㄱ ㄱ ㅗ ㅏ ㄱ` / `사과` → `ㅅ ㅏ ㄱ ㅗ ㅏ`
- 시도 **5번**. 입력은 **전용 화면 자판만** (물리 키보드/IME 미사용).
  - 단일 자모만 노출 — 자음 14 + 모음 10. 쌍자음/복모음/겹받침 키 없음.
- **자모 분해 규칙** (키보드 24자모로만 환원, SSOT = `lib/jamo-mapping.ts`):

  | 표준 자모 | 게임 분해 |
  |---|---|
  | ㄲ ㄸ ㅃ ㅆ ㅉ | ㄱㄱ / ㄷㄷ / ㅂㅂ / ㅅㅅ / ㅈㅈ |
  | ㅐ ㅒ ㅔ ㅖ ㅢ | ㅏㅣ / ㅑㅣ / ㅓㅣ / ㅕㅣ / ㅡㅣ |
  | ㅘ ㅚ ㅝ ㅟ | ㅗㅏ / ㅗㅣ / ㅜㅓ / ㅜㅣ |
  | ㅙ ㅞ | ㅗㅏㅣ / ㅜㅓㅣ (3자모) |
  | 겹받침 (ㄳ ㄵ ㄶ ㄺ …) | 두 자음으로 분해 |

- **채점** (분해된 자모 시퀀스 위치별):
  - 🟩 자모+위치 정답 / 🟨 자모는 있으나 위치 다름 / ⬜ 없음
  - 중복 자모는 **정답에 존재하는 개수만큼만** 색칠 (본가 2-pass 규칙)
  - **자판 색상 누적**: 키에도 🟩🟨⬜ 반영, 우선순위 🟩 > 🟨 > ⬜
- **종료**: 5번 내 정답 → 승리 / 5번 실패 → 패배(정답 음절 공개)
- 진행 중 표시는 **자모만**, 종료 모달에서 음절 공개.

---

## 3. 단어 사전

**정답 풀 ⊂ 입력 허용 풀** 2단계 (본가 방식: 정답은 좁고 깔끔, 입력은 넓게).

```
klue/bert-base vocab → 자모 개수별 분리(words-by-jamo/) → 5자모 추출
  → 입력 허용 풀(guesses): 5자모 전부
  → 정답 풀(answers): 표제어(명사+고유명사) ∩ 빈도 ∩ 충돌 제거
```

- 산출물 `src/data/words.json = { answers: {syllable, jamo}[], guesses: string[] }`
- 무효 단어 제출: 시도 차감 X, 행 흔들림 + 토스트
- 재생성: `npm run build:dict && npm run build:words`

## 4. 결과 공유 / 통계

- **공유**: 🟩🟨⬜ 그리드 + 시도/소요시간 + 현재 링크를 **OS 기본 공유 시트**(`navigator.share`)로. iOS/Android 지원, 데스크톱은 클립보드 복사 폴백.
- **통계 (현재)**: 라운드 종료 시 **localStorage**(`kordle.plays`)에 저장. 총 도전/정답률/평균 시도·시간/분포/최근 기록. 수동 초기화.
- **통계 (추후)**: Supabase `plays` 테이블로 승격 → 크로스 디바이스 + 친구 리더보드. 로컬 필드를 DB 스키마와 1:1로 맞춰둠 → 마이그레이션 용이. (스키마 [ERD.md](./ERD.md))

---

## 5. 인증 (구현 완료)

- **Provider**: 카카오 OAuth 단일, **Supabase Auth** (`@supabase/ssr`)
- 동의항목: `profile_nickname`(필수), `profile_image`(선택). `account_email`은 선택(비즈 앱 필요) — **이메일 없는 유저 허용** ON.
- 유저 식별: `auth.users.id`(UUID) + `provider_id`(카카오 회원번호). **이메일은 식별키 아님.**
- 세션: 미들웨어로 SSR 쿠키 갱신, `/auth/callback`에서 code→세션 교환.
- 헤더에 로그인/프로필(닉네임)/로그아웃. 환경변수 미설정 시 버튼 숨김(게임은 정상).

### 6.1 진입 플로우

```
랜딩 ( / )
├─ [혼자 연습]  → /play   (기존 게임, 무한, 로그인 X)
└─ [같이 풀기]  → 로그인 필요
                  ├─ 미로그인 → 카카오 로그인 모달 → 로그인 후 진입
```

- 방 클릭 → 방 상세 (이번 범위 밖)

### 6.3 라우트 (제안)

## 7. 기술 스택

- **Framework**: Next.js 16 (App Router) + TypeScript + Tailwind v4
- **상태**: React useReducer (순수 게임 상태 머신)
- **저장소**: localStorage(진행/통계) · Supabase(인증, 추후 방/통계)
- **인증/DB**: Supabase (Auth + Postgres + RLS), Kakao OAuth
- **테스트**: Vitest (lib 47개) · **배포**: Vercel
- **보안**: anon 키는 공개(RLS가 보안 경계), service_role/카카오 시크릿은 **커밋 금지**, `.env*.local` gitignore
